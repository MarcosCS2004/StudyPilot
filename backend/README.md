# StudyPilot SaaS

Plataforma de aprendizaje adaptativo con RAG y Spaced Repetition (SM-2).

## Estructura

- **backend/**: API FastAPI
- **frontend/**: React/Next.js
- **infra/**: Azure Bicep
- **docs/**: Documentación

## Quick Start

```bash
cp .env.example .env
cd backend && pip install -r requirements.txt
python main.py
```
